💡 提交指南

如何提交你的作品

  1. 将所有网页文件、图片、CSS文件放在同一个文件夹中
  2. 文件夹命名为：班级_姓名_网页展活动
  3. 将文件夹压缩为ZIP格式
  4. 点击"提交作品"按钮上传压缩包
  5. 等待老师批改和评语

注意事项：

  • 确保所有文件路径正确，使用相对路径
  • 图片文件名不要使用中文和特殊字符
  • 确保首页文件名是 index.html
  • 有问题及时向老师请教
