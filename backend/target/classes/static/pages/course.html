3.1 Dreamweaver界面与工具使用

Dreamweaver是Adobe公司推出的专业网页制作工具，功能非常强大！


Dreamweaver主要界面区域：
  • 菜单栏 - 文件、编辑、查看等命令
  • 工具栏 - 常用工具快捷按钮
  • 文档窗口 - 编辑网页的主要区域
  • 属性面板 - 设置选中元素的属性
  • 文件面板 - 管理网站文件

三种视图模式：
  • 代码视图 - 直接编写HTML/CSS代码
  • 设计视图 - 所见即所得的可视化编辑
  • 拆分视图 - 同时显示代码和设计

4.1 网页布局与美化技巧

好的布局能让网页看起来更专业、更美观！


常用布局方法：
  • 表格布局 - 适合简单的规整布局
  • DIV + CSS - 现代网页的主流布局方式
  • Flexbox - 弹性盒子布局
  • Grid - 网格布局

美化小技巧：
  • 选择和谐的配色方案
  • 适当使用圆角和阴影
  • 保持足够的留白空间
  • 图片和文字搭配得当
  • 添加一些动画效果
